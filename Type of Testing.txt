There are different stages for manual testing such as unit testing, integration testing, system testing, and user acceptance testing. Testers use test plans, test cases, or test scenarios to test a software to ensure the completeness of testing.

UNIT TESTING is a level of software testing where individual units/ components of a software are tested.
It is conducted by independent software testers
It is a type of WhiteBox testing

INTEGRATION TESTING is a level of software testing where individual units are combined and tested as a group.
It is conducted by group of programmers
It is a type of BlackBox testing

What is System Testing?
System testing is the execution of the software in its final config.

System testing is conducted developer.

It is a type of black box testing.

What is Acceptance Testing?
Acceptance Testing is to evaluate and validate that the system's funcionalty meets the requirements before final delivery.

Acceptance Testing is conducted by customers and end-users.

It is a type of black box testing.
